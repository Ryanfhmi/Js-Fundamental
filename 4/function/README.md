# Materi Function (Folder)

Halaman ini berisi materi lengkap tentang function di JavaScript yang ada di `index.html`.

Cara pakai:
- Buka `index.html` di browser.
- Gunakan tombol "Jalankan" untuk melihat output contoh.
- Gunakan tombol "Lihat Penjelasan" untuk membuka solusi.
- `script.js` berisi logika interaktif; `style.css` untuk gaya.

Contoh tambahan ada di folder `examples/`.

Catatan keamanan: contoh menggunakan `new Function(...)` untuk mengeksekusi potongan kode di halaman ini. Jangan menjalankan kode yang tidak dipercaya di lingkungan produksi.
